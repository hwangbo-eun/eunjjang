# Todo List

투두리스트 앱을 만들기 전에 요구사항과 개발 계획을 정리하는 폴더입니다.

## 구성

- `plan.md`: 계획 에이전트 지침과 투두리스트 앱 요구사항/개발 계획
- `implementation-agent.md`: 클린 아키텍처 기반 구현 에이전트 지침

## 사용 방법

1. `plan.md`를 보면서 요구사항을 구체화합니다.
2. 사용자의 답변을 바탕으로 `plan.md`를 계속 업데이트합니다.
3. 요구사항이 충분히 정리되면 `implementation-agent.md`를 기준으로 구현합니다.
4. 이 폴더 안에서 바로 구현하거나, 별도 구현 폴더를 만들고 개발을 시작합니다.
